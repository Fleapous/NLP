This readme conserns following files: scrapper.py, NLP_Final.ipynb, LIVE DEMO.ipynb, main.py, nlp_module.py

To run listed scripts following libraries and their dependencies should be installed (preferrably using Anaconda) 


1. BeautifulSoup:
	pip install beautifulsoup4

2. Pandas:
	pip install pandas

3. word2number:
	pip install word2number

4. spacy:
	please follow the guide from the official documentation
	https://spacy.io/usage/embeddings-transformers#transformers
